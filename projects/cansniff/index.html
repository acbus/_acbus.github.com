CANsniff

Sniffs the CAN bus

Introduction

This is a project made for sniffing CAN protocol bus. The complete source code and hardware designs are open source. CANsniff can handle upto bitrates of 1 Mbits/s. The initial paln is to use STM32F4DISCOVERY . A hardware will be designed once the software is ready.
