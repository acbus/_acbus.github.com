CANsniff

Open Source Hardware based CAN protocol analyser

Introduction

This is a project made for sniffing CAN protocol bus. The complete source code and hardware designs are open source. CANsniff can handle upto bitrates of 1 Mbits/s. The initial paln is to use STM32F4DISCOVERY as hardware. The software development will be done on this hardware and once it is done, a new hardware will be designed.

Hardware design

1) Serial port for debugging

2) USB for data transfer to PC

3) SD card support for standalone operation 

Software design

The default serial port in the controller can be used for debugging purpose and intial development. For hight speed data transfer, the USB port can be used.
