CANsniff

An Open Source Hardware based CAN Protocol Analyser



Introduction

CANsniff is an open source CAN protocol analyser for sniffing CAN protocol bus. CANsniff aims to design a hardware for analysing the CAN bus traffic and a software on the host(PC) side. The complete source code and hardware designs are open source. CANsniff can handle upto bitrates of 1 Mbits/s. The initial paln is to use STM32F4DISCOVERY as the hardware. The software development will be done on this hardware and once it is done, a new hardware will be designed.


Hardware design

1) Serial port for debugging

2) USB for data transfer to PC

3) SD card support for standalone operation 


Software design

The default serial port in the controller can be used for debugging purpose and intial development. For hight speed data transfer, the USB port can be used.
